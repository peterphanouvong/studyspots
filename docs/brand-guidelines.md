# Study Spots — Brand Guidelines (for Forms)

Use this to style the three external forms (Tally / Formspree / Google Forms) so they
feel like a seamless part of the app. Most builders let you set an accent colour, font,
background, and button style — match the values below.

---

## Brand in one line

**Study Spots** helps students and remote workers answer one question: *"Can I actually
study here?"* The brand is **clean, trustworthy, warm, and unfussy** — Airbnb-calm, not
flashy. Forms should feel quick and friendly, never bureaucratic.

## Wordmark

> **Study Spots** + a small **Deep Teal dot** (·) trailing the name — it reads as a
> location pin.

- Always "Study Spots" in **Soft Black**, dot in **Deep Teal**.
- If the form builder allows a logo/header, use the wordmark; otherwise just the name in
  the heading is fine. Don't recolour or stretch it.

## Colours

| Role | Hex | Use in forms |
|---|---|---|
| **Deep Teal** (primary) | `#0A695B` | Buttons, accent, links, focused inputs, selected options |
| **Teal (hover/pressed)** | `#085449` | Button hover/active |
| **White** | `#FFFFFF` | Form background / canvas |
| **Surface** | `#F9FAFB` | Input fields, cards, section blocks |
| **Primary text** | `#101828` | Questions, headings, answers |
| **Secondary text** | `#717171` | Helper text, placeholders, descriptions |
| **Border** | `#E5E7EB` | Input outlines, dividers (keep subtle) |

Teal is the **only** accent colour. Everything else is white + greys. No gradients, no
second accent.

## Typography

- **Font:** **Plus Jakarta Sans** (Google Fonts). If the builder can't load it, fall back
  to a clean geometric sans (e.g. system UI), never a serif.
- **Headings / questions:** weight **600–700**, slightly tight letter-spacing.
- **Body / answers:** weight **400–500**.
- **Helper text:** weight **400**, in secondary grey `#717171`.

## Geometry & feel

- **Rounded, friendly.** Inputs and buttons: pill or `12px`+ radius. Cards/sections:
  `16px`.
- **Minimal shadows.** Rely on the light grey surface for separation, not heavy borders
  or drop shadows.
- Generous spacing; one question per "thought." Don't crowd.

## Buttons

- **Primary (submit):** Deep Teal `#0A695B` background, **white** text, weight **600**,
  pill shape. Hover → `#085449`.
- **Secondary (if any):** white background, Soft Black text, subtle `#E5E7EB` border.

## Voice & tone

- **Warm, plain, brief.** Talk like a helpful local, not a corporation.
- Lowercase-friendly, contractions welcome ("we'll", "you're").
- No jargon, no "kindly", no marketing fluff.
- Labels are short and human: *"Which cafe?"* not *"Please specify the establishment."*
- Reassure on anything optional: *"Optional — only if you'd like a reply."*

**Examples**
- Title: *"Know a spot we're missing?"* / *"Tell us what you think"* / *"Is this your cafe?"*
- Helper: *"Roughly where is it? A suburb is enough."*
- Confirmation: *"Thanks! We'll check it out."* — friendly, short, no exclamation overload.

## Do / Don't

| Do | Don't |
|---|---|
| One accent (teal) on white + grey | Add extra colours or gradients |
| Plus Jakarta Sans, rounded fields | Serif fonts, sharp corners |
| Short, human questions | Long forms or formal wording |
| Keep most fields optional | Gate submission behind lots of required fields |
| Match the app's calm, clean feel | Heavy borders, busy backgrounds, clip-art |
